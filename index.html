King and Jenn

Wedding website.

This was a website I made for some of my good friends. I wanted them to have full control over the design, so I built the entire site from the ground up using no frameworks. Majority of the website was built with HTML, CSS, JavaScript/jQuery, PHP, and SQL.

The couple wanted to shy away from the conventional RSVP process so I developed a solution that would allow guests to visit the website and submit their information via the RSVP page. The backend functionality was built using PHP and a MySQL server while I utilized jQuery to dynamically display the appropriate content for each user.

Ninth Rock

Video Game Project

Ninth Rock was a game that I developed during my semester-long internship with TimeGate Studios. I was one of five interns on the team, each of us specializing in a different part of the project. As the lead art director, I was in charge of creating the main character models, along with some of the other assets seen throughout levels.

After seeing that Pluto had been demoted to a dwarf planet in the news, I came up with the storyline concept for the project. Everybody thought it was a neat idea so we went with it. The main objective is to sneak through MASA (you guessed it, a facility similar to NASA) in order to find the growth ray gun, fire it at Pluto, and have the planet grow large enough to be reinstated as a real planet.

We had the pleasure of using TimeGate's state-of-the-art motion capture facility to record my real-life movements and map them onto the characters. This was SO MUCH FUN and made for an unforgettable experience. It took a lot of work to get things perfect, but it definitely paid off, and I am extremely proud of the results.

Technical Information

  • Game engine: Unreal Development Kit
  • Models/Animation: 3D Studio Max & MotionBuilder
  • Textures: Photoshop
